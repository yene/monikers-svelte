# Monikers Svelte

[Monikers](http://www.monikersgame.com) is a party game which can be played on one phone.


## Get started

Install the dependencies...

```bash
cd monikers
npm install
```

...then start [Rollup](https://rollupjs.org):

```bash
npm run dev
```
